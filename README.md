# Treasure Labyrinth

A text-based adventure game where you explore a mysterious labyrinth, solve riddles, and find treasure!

## Installation

1. Install Poetry: `pip install poetry`
2. Install dependencies: `make install`
3. Run the game: `poetry run treasure-labyrinth`

## How to Play

Use commands like `go north`, `look`, `take item`, `use item`, `inventory`, `solve answer`, and `quit`.

## Development

- `make project` - Setup project
- `make lint` - Run linter
- `make build` - Build package
